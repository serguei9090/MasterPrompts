# 🎨 Frontend Architecture

## Component Hierarchy (Atomic Design)

- **Atoms**: Smallest UI units (buttons, inputs).
- **Molecules**: Combined atoms (search bars, form fields).
- **Organisms**: Complex sections (headers, navigation, sidebars).
- **Templates**: Page-level layout structures.
- **Pages**: Individual view instances.
