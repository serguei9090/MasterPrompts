# AI Memory System

This directory contains the persistent memory of the Antigravity ecosystem.
It is structured to be "AI-Optimized", prioritizing high signal-to-noise ratio and fast indexing.

## Structure
- `index.md`: Central map of all memories.
- `decisions/`: Architecture Decision Records (ADRs).
- `entities/`: Knowledge about tools, people, and specific technologies.
- `patterns/`: Repeated implementation patterns.
- `lessons/`: Historical "Gotchas" and solutions.
- `sessions/`: Distilled summaries of important conversation/work sessions.
