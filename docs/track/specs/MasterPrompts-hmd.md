# Spec: Create test folder and simple python script

## Bead ID: MasterPrompts-hmd

## Overview
Create a new directory named `test/` at the project root and implement a simple Python script `test/simple_script.py` containing 5 specific functions.

## Requirements
- Directory: `test/`
- Script: `test/simple_script.py`
- Functions:
  1. `get_random_number()`: Returns a random integer.
  2. `print_hello(name)`: Prints "hello {name}".
  3. `add(a, b)`: Returns the sum of a and b.
  4. `multiply(a, b)`: Returns the product of a and b.
  5. `square(n)`: Returns the square of n.

## Implementation Plan
1. Create the `test/` directory.
2. Create `test/simple_script.py`.
3. Implement the functions using standard Python libraries (e.g., `random`).
4. Add basic docstrings following the project's standards.

## Verification
